# alfredo.blog Theme

Custom Hugo theme for [alfredo.blog](https://alfredo.blog) on Micro.blog.

## Structure

- `layouts/` - Hugo templates
- `static/` - CSS, JS, fonts
- `config.json` - Theme configuration
- `plugin.json` - Plugin metadata
- `theme.toml` - Theme metadata

## Workflow

1. Edit templates locally
2. Push to GitHub
3. In Micro.blog: Design → Edit Custom Themes → click **Reload** to pull updates
